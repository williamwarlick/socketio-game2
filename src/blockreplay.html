Game Instructions

  1. You will watch a series of 9 games.
  2. Each game you will see someone pursure a some goal.
  3. At some point during the game, you will be asked to describe what you think the goal is.
  4. Blocks can be placed on top of other blocks or on the floor of the grid.
  5. If a block has another block on top of it, it cannot be moved until the block on top of it (i.e., covering it) is moved.
  6. Include more info about differnt game, how it could be stopped whenever?

The game will automatically advance to the next round once a goal is achieved.

If you think that you have achieved a given goal but the game is not advancing, please refresh the page.
